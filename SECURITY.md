# Security Policy

## Supported Versions

The following table lists the versions of the AND-CP platform that are actively supported with security patches:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you find a security bug or potential vulnerability in the AND-CP platform, please do not open a public Github issue. Instead, report it directly to our security response team:

1. Email your findings to `security@andcp-project.org`.
2. Include a detailed description of the vulnerability, step-by-step instructions to reproduce it, and any proof-of-concept scripts or configurations.
3. We will acknowledge receipt of your report within 48 hours and work with you to patch the vulnerability before public disclosure.
