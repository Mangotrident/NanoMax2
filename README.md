# NanoMax2

A real-time EEG decoding and adaptive brain–computer interface system built on BCI Competition IV motor imagery datasets, featuring deep learning models, classical signal processing pipelines, few-shot calibration, and a live neuro-control dashboard.

AND-CP is a full-stack neurotechnology platform that translates EEG brain signals into predicted motor intent in real time.

It combines:

Classical BCI signal processing (CSP, bandpower, Riemannian geometry) Deep learning EEG models (EEGNet, ShallowConvNet, DeepConvNet) Transfer learning for subject adaptation Few shot calibration for new users Real time streaming inference engine Interactive neuroscience control dashboard

The system is designed to simulate and study assistive decoding for motor-impaired users, with a focus on Cerebral Palsy like signal variability.
